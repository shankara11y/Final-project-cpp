# Digital Notice Board System (C++)

## Overview
The Digital Notice Board System is a simple C++ console application that allows users to manage notices digitally.  
The system provides basic operations such as adding, deleting, and viewing notices through a menu-driven interface.

This project demonstrates the use of basic programming concepts such as arrays, classes, loops, and functions.

---

## Features
- Add a new notice (Title and Description)
- Delete an existing notice
- View all notices
- Menu-driven user interface

---

## Technologies Used
- Programming Language: C++
- Concepts Used:
  - Classes
  - Arrays
  - Loops
  - Conditional statements
  - Functions

## How to Run the Program

### Step 1: Compile the program

g++ main.cpp -o notice


### Step 2: Run the program

./notice


The program will display a menu where you can:
1. Add Notice
2. Delete Notice
3. View Notices
4. Exit
